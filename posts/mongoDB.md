# SQL vs NoSQL
- SQL are Relational Databases whereas, NoSQL are non-relational or distributed database.
**Popular SQL databases**
- MySQL
- PostgreSQL
- SQLite

**Popular NoSQL Databases**
- MongoDB
- Redis
- Amazon DynamoDB

## MongoDB
- A popular NoSQL database
- Datas are stored in the form of key-value pair.

```
	{
		title : string
		required : true 
	}
```
